🧩 카피페 위젯 생성기 (구분선 포함)

값을 입력하면 오른쪽에 미리보기와 HTML 코드가 실시간 갱신됩니다.

🎨 디자인 미세조정
🖼️ 배경 이미지 (URL)
👤 프로필 이미지
📚 카피페 데이터
※ 두 번째 화자가 오른쪽 말풍선.
프로필 이미지 사용 시 각 캐릭터 URL 입력.
✨ 대사에 이미지 URL을 넣으면 자동으로 이미지로 표시됩니다.
📏 구분선은 '구분선 추가' 버튼으로 삽입 가능, 텍스트 선택사항.
🖥️ 실시간 미리보기 (3:4)
버튼/랜덤 동작 확인
📋 생성된 HTML 코드 (티스토리용) 

    💡 코드를 복사하여 티스토리 HTML 모드에 붙여넣으세요.
※ 기본모드 전환 시 코드가 깨질 수 있으니 주의!
